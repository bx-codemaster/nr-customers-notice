Kundeninformations-System! Aktuelle Informationen direkt im Shop anzeigen.
Mit dieser Erweiterung besteht die Möglichkeit den Kunden im Shop zu informieren. 
Der Hinweis kann auf einen bestimmten Zeitraum begrenzt werden und/oder 
auf nur für spezische Kundengruppen und/oder 
nur auf bestimmten Seiten angezeigt werden und/oder 
nur Kunden die noch kein Newsletterempfänger sind. 
Mit Hilfe verschiedener Templates können die Hinweise unterschiedlich angezeigt werden.
